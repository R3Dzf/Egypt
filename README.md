# Egypt World Cup 2026 Path Calculator

موقع بسيط لحساب وضع مصر في كأس العالم 2026 ومين ممكن تواجه في دور الـ32.

## أهم المميزات

- يحدث نتائج دور المجموعات من ESPN public scoreboard endpoint عند فتح الصفحة أو عند الضغط على تحديث.
- يحسب ترتيب كل مجموعة وأفضل 8 ثوالث.
- يحاول تحميل جدول Annex C لتحديد خصم مصر بدقة حسب تركيبة أفضل الثوالث.
- يحدد طريق مصر إذا كانت أول/ثاني/ثالث المجموعة G.
- يعرض قسم **الخصوم المحتملين لمصر الآن**: يجرب نتائج المباريات المتبقية ويجمع كل الطرق الممكنة والخصوم المحتملين بناءً على الوضع الحالي.
- يسمح بتعديل النتائج يدويًا وتجربة سيناريوهات مثل فوز/تعادل/خسارة مصر أمام إيران.

## التشغيل محليًا

### الطريقة السهلة
افتحي `index.html` مباشرة في Chrome أو Edge.

### لو التحديث الأونلاين لم يعمل مع الفتح المباشر
افتحي Terminal / CMD داخل فولدر المشروع وشغلي:

```bash
python -m http.server 8000
```

ثم افتحي:

```text
http://localhost:8000
```

ولو `python` غير معروف على Windows، جربي:

```bash
py -m http.server 8000
```

## طريقة قراءة قسم الخصوم المحتملين

- لو عدد المباريات المتبقية قليل، الموقع يحسب كل السيناريوهات الممكنة بنموذج فوز/تعادل/خسارة.
- لو عدد المباريات المتبقية كبير، الموقع يستخدم عينة كبيرة من السيناريوهات حتى لا يعلق المتصفح.
- النسب المعروضة ليست احتمالات مراهنات، لكنها نسبة ظهور كل طريق داخل السيناريوهات المحسوبة أو العينة.

## النشر أونلاين

أسهل طريقتين:

1. GitHub Pages:
   - ارفعي الملفات إلى Repository جديد.
   - Settings → Pages → Deploy from branch → main / root.

2. Netlify:
   - اسحبي فولدر الموقع إلى Netlify Drop.

## ملاحظات

- ESPN endpoint غير موثق رسميًا، لذلك ممكن يتغير أو يتوقف. لهذا يوجد تعديل يدوي.
- عند التعادل الكامل في النقاط وفارق الأهداف والأهداف، الموقع يضع تنبيهًا لأن FIFA تستخدم tiebreakers إضافية مثل fair play وFIFA ranking.
- حساب الخصوم المحتملين يعتمد على النتائج الحالية والنتائج التي تدخلينها يدويًا.
